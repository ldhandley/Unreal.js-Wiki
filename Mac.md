## For a fresh start
1. Create a new C++ project
1. Open terminal
```
# install v8
brew install v8
cd ~/Documents/Unreal\ Projects/<your project>

# download plugin
git clone https://github.com/ncsoft/Unreal.js-core Plugins/UnrealJS

# build and run!
open <your project>.uproject
```

* There is no need to see XCode running
